## 🚀 如何使用

### 快速開始

```bash
# 啟動獎勵配置GUI
python demo_reward_config_gui.py
```

### 在現有程序中使用

```python
from simulation_controller import SimulationController
controller = SimulationController(trained_agent=your_agent)
controller.run()  # 會包含獎勵配置界面
```
